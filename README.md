 Neon Biometric Lock 

A visually stunning interactive web page simulating a "biometric lock" experience with:

- Animated galaxy background 
- Typewriter-style personalized messages 
- Heart animations 
- Neon glow card and cinematic vignette effects
- QR code reveal for added flair 
- Responsive and mobile-friendly design

 How It Works
1. User enters the access key (`omoh`) and clicks "SCAN".
2. Access granted triggers:
   - Music playback 
   - Typewriter message with romantic/poetic flair
   - Heart animations floating upwards
   - QR code display
   - Glowing neon card and vignette cinematic effect

Tech Stack
HTML5 & CSS3 (with CSS variables & animations)

Vanilla JavaScript for animations & interactions

Audio and canvas effects for immersive UI
